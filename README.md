# Color Game 🎨

A fun and interactive Android color game developed as a school project. 

## 📝 Description

Color Game is a fast-paced reaction mobile game designed to test cognitive speed. Players are challenged to identify the physical color of the text displayed on the screen rather than reading the word itself (based on the Stroop effect). 

This project was built to demonstrate core mobile programming concepts, UI design, and event handling on Android using Xamarin and C# as part of an academic requirement.

## ✨ Features

* **Cognitive Challenge:** Randomized text and color generation designed to test reaction time.
* **Score Tracking:** Keeps track of the player's highest score during the active session.
* **Timer Mechanic:** A countdown timer that adds pressure to each round.
* **Touch Interface:** Clean, intuitive, and responsive mobile design built for Android.

## 🛠️ Tech Stack

* **Language:** C#
* **Framework:** Xamarin (Xamarin.Android / Xamarin.Forms)
* **IDE:** Visual Studio
